# Banana Prompts PHP SDK

This is the PHP SDK for [Banana Prompts](https://bananaprompts.org).

## Features
- Search through a massive library of Midjourney & Stable Diffusion prompts.
- **Image-to-Prompt:** Reverse engineer any image into a text prompt.
